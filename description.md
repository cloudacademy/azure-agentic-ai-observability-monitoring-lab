Microsoft Foundry Agent Service enables developers to build, deploy, and manage AI agents that can interact with users and other services. To ensure the reliability and performance of these AI agents, it is crucial to implement robust observability and monitoring solutions.

Generative AI Operations (GenAIOps) is a set of practices and tools designed to manage and monitor generative AI models and applications effectively. By leveraging GenAIOps, organizations can ensure that their AI agents operate efficiently, deliver high-quality results, and maintain compliance with regulatory requirements.

### Learning objectives

Upon completion of this beginner-level lab, you will be able to:

- Create an AI agent using Microsoft Foundry Agent Service
- Implement observability and monitoring solutions for AI agents
- Utilize Application Insights to track AI agent performance and user interactions
- Set up alerts and notifications for AI agent anomalies

### Intended audience

- Cloud Architects
- AI Engineers and Developers
